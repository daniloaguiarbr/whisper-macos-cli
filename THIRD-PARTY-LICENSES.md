## External Binaries (Optional)

### ffmpeg

- License: LGPL 2.1+ or GPL 2+ depending on build configuration
- Copyright: FFmpeg project contributors
- Website: https://ffmpeg.org
- Used by: whisper-macos-cli v0.1.2+ for video container audio
  extraction and OGG/Opus fallback
- Notes: ffmpeg is NOT linked into the binary. It is invoked as a
  subprocess only when needed. If ffmpeg is not installed, the
  CLI returns exit code 69 with a clear install hint.

## Regeneration

To regenerate this document after dependency changes:

```bash
cargo install cargo-about --locked
cargo about generate about.hbs -o THIRD-PARTY-LICENSES.md
```

The `about.hbs` template lives at the repository root.
