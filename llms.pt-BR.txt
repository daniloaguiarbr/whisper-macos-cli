# whisper-macos-cli

> CLI exclusiva para macOS para transcrição local de áudio via whisper.cpp com aceleração Metal GPU

whisper-macos-cli é uma CLI Rust que transcreve arquivos de áudio
para texto usando whisper.cpp com Metal GPU da Apple Silicon. Aceita
áudio via argumentos de arquivo ou pipe stdin e emite JSON no
stdout. Projetada para integração com agentes de IA via contrato
estrito stdin/stdout JSON.

## Características Principais

- Aceleração Metal GPU em Apple Silicon (M1/M2/M3/M4)
- Qualidade máxima por padrão: modelo large-v3, BeamSearch beam_size=8
- Suporta: MP3, OGG/Vorbis, OGG/Opus (WhatsApp), FLAC, WAV, AAC
- Detecção automática de idioma a partir do locale do macOS
- Chunking baseado em VAD previne alucinações
- Modo NDJSON para processamento em lote
- correlation_id UUID v7 em cada saída para rastreabilidade
- schema_version em cada saída para evolução segura
- Downloads de modelo apenas HTTPS com identificação User-Agent
- Local apenas: zero telemetria, zero chamadas em nuvem

## Início Rápido

```bash
cargo install whisper-macos-cli
whisper-macos-cli models download
whisper-macos-cli transcribe audio.ogg
```
